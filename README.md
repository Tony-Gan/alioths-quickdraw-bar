# 模组说明

### 兼容性
* **验证版本**：Foundry VTT 13 + D&D 5.2.x
* **语言支持**：仅支持中文

---

### 模组简介
本模组提供一个高度可定制化的快捷动作栏（HUD），用于快速调用角色卡或 Token 的各项功能。旨在让玩家无需频繁打开角色卡即可完成绝大部分操作。部分设计参考了 [Larkinabout 的 Token Action HUD](https://github.com/Larkinabout/fvtt-token-action-hud-dnd5e)。

### 启动方法
模组安装并启用后，Foundry VTT 界面左侧的“Token 控件”菜单中会出现一个闪电图标。点击该图标即可开启或关闭快捷栏。

### 功能与使用说明

#### 1. 界面交互
快捷栏默认位于屏幕下方。拖动窗口标题栏可自由移动位置，拖动右下角可调整界面缩放比例。双击标题栏可将动作栏最小化，再次双击即可复原（该功能由FVTT提供）。

#### 2. 角色绑定
快捷栏会自动绑定当前用户拥有权限且在场景中存在的角色 Token。若当前场景中有多个属于你的角色，点击场景中的 Token 可自动切换动作栏的绑定目标。你也可以通过快捷栏左上角的下拉菜单手动锁定特定角色。快捷栏内的按钮状态（如属性值、资源消耗、法术位）与角色卡数据实时同步。

#### 3. 物品与功能管理
动作栏包含所持物、特性、法术等多个标签页，提供丰富的管理功能：
* **悬停提示**：将鼠标悬停在物品或法术按钮上，会自动弹出悬浮窗，显示详细描述、使用次数和前置要求。
* **右键菜单**：右键点击任意物品按钮可呼出功能菜单。支持仅在动作栏上修改按钮名称，不影响角色卡数据（适合简化过长的物品名）。支持将常用物品加入“收藏”标签页，方便快速调用。支持将不常用的物品隐藏。如需找回，可通过顶部下拉框调整“显示隐藏物品”选项。

#### 4. 快捷掷骰
在点击栏按检定/豁免标签栏中的按钮时，按住辅助键可快速调整掷骰模式，无需并在对话框中确认：
* **Shift + 点击**：快进掷骰（直接投掷普通结果）
* **Alt + 点击**：优势掷骰
* **Ctrl + 点击**：劣势掷骰

#### 5. 法术与状态
法术标签页中，按钮右侧显示 `×` 标记的表示该法术“未准备”。在“状态/移动”标签页，支持两种点击模式。
**左键点击**：切换常规状态图标。
**右键点击**：切换状态覆盖图标。
