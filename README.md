# 模组说明

### 兼容性
* **验证版本**：Foundry VTT 13 + D&D 5.2.x
* **语言支持**：仅支持中文

---

### 模组简介
本模组提供一个高度可定制化的快捷菜单，用于快速调用角色卡或 Token 中的各项功能。模组的部分设计与排版参考了 [Larkinabout 的 Token Action HUD](https://github.com/Larkinabout/fvtt-token-action-hud-dnd5e)。

### 使用方法
模组安装并激活后，在 Foundry VTT 界面左上角的“Token 控件”菜单中会出现一个闪电形状的入口。点击该图标即可打开模组交互界面。
